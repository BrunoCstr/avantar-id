# Guia do Usuário – Sistema 2FA Avantar

Bem-vindo ao sistema de autenticação em duas etapas (2FA) da Avantar! Este guia vai te ajudar a usar o sistema de forma fácil e segura.

## O que é o sistema 2FA?

O sistema 2FA serve para proteger ainda mais o seu acesso. Além da sua senha, você vai precisar de um segundo código para entrar. Assim, mesmo que alguém descubra sua senha, não conseguirá acessar sua conta sem esse código extra.

## Como acessar o sistema

1. **Acesse a página de login:**  
   Abra o site normalmente e coloque seu e-mail e senha.

2. **Receba o código:**  
   Após realizar o pedido do envio no portal da seguradora, acesse o sistema e clique no botão 'Atualizar e-mail'. Esse botão será responsável por verificar a caixa de entrada do e-mail onde os códigos são recebidos, e preencherá automaticamente o código mais recente no campo.

4. **Pronto!**  
   Se o código estiver certo, você entra normalmente no sistema.

## Dicas importantes

- **NUNCA compartilhe seu código e seu acesso a este sistema:**  
  O sistema trata dados sensíveis, jamais compartilhe seu acesso.

- **Se não receber o código:**  
  É possível que o código ainda não tenha chegado à caixa de entrada devido a uma instabilidade no sistema da seguradora. Recomendamos reenviar o código ou aguardar alguns instantes e clicar novamente em 'Atualizar e-mail' no sistema. Caso o problema persista e o código não seja atualizado, por favor, abra um chamado para a Tecnologia para que possamos investigar.

## Sobre a área do usuário

Após acessar o sistema, você terá acesso à área compartilhada dos franqueados, onde encontrará os códigos de acesso das seguradoras, tanto para o acesso treino quanto para o master.

Se você já possui um acesso individual, precisará apenas dos códigos das seguradoras com acesso master, que são identificadas pelo nome normal sem parênteses (por exemplo: (TREINO) entre parênteses indica que é um acesso TREINO e pode ser desconsiderado caso já possua acesso individual).

## Segurança

- Sempre saia do sistema ao terminar de usar, principalmente em computadores compartilhados.
- Não compartilhe seu acesso com ninguém.

## Suporte

Se tiver qualquer dúvida ou problema, entre em contato com a Tecnologia.
